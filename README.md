## About

**e_store** is a full-featured online store application designed for scalability and customization. The app leverages modern JavaScript (ES6+) best practices for both front-end and back-end development.

## Features

- Product catalog and search
- Shopping cart with add/remove functionality
- User authentication and registration
- Order processing and payment simulation
- Responsive UI

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (recommended version 18+)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
git clone https://github.com/Ravshanjohn/e_store.git
npm run build
```

### Running the Application

```bash
npm run start
```


## Scripts

- `start`: Runs the application in production mode.
- `dev`: Starts the development server with hot reloading.
- `test`: Runs the test suite.

Update the `package.json` file to customize or add more scripts.
